回溯(二)人类孤独而平凡而伟大

我不太算得上是一个音乐发烧友，但遇到很喜欢的专辑，也逐渐有了去买一张 CD 的想法，我的备忘录里专门有这样的一张列表，用来记录我可能会购买的专辑。

れをる的《No Title-》我一直挺喜欢的，在我看来质量很高的一张专辑。不知为何，鬼使神差地没有去日本亚马逊上找——过去我要买 CD 的第一反应总是打开 Amazon JP ——我很自然地打开淘宝寻觅着这张专辑。让我颇感意外的是，虽然れをる在国内有一定的人气，但淘宝上的实体专辑并不多，反复筛选了之后，我暂时敲定了两家，一家在 165¥ 的售价下拥有三笔月销，而且一位买家还留下了“老板人真好”的评论，另一家便宜 10 元钱，但是没有月销记录也没有评论，我决定都和各自客服聊一聊，再做打算。

我先点开了售价较低的一家——店铺名叫“团子代购”——询问店主这张专辑是现货还是需要预订，店主回复还是比较及时的，说明了需要预订，我匆匆回复了一句就结束了对话，可能是我多余的顾虑，总觉得店主最后一句“不客气”带着一丝失望的语气。犹豫了一会儿，我点开这家店铺的详情，打算看看其中的商品——大多数是原版的 CD 或书籍，不少精品。如今这种能依旧保持着对于二次元文化的热衷并依靠并不算稳定的代购来营生的店家，我多少还是有些好感的，毕竟 ACG 这样一个那么小的圈子里走出的人，能够在诺大的世界中相遇，难道不是一种意义上的奇迹吗？

“贵店买的 CD 应该是记销量的吧？”我又问了一句
“是的，记。”

我大概不会下单，这么想着，还是把这家店铺放进了收藏夹

和另一家店铺的店主交流的过程，则是另一种形式的经历了，也是在问完能否有现货之后，我放下手机忙起自己的事情，等待店主的回复，几分钟后，手机噔噔响了一下——“后天就能发现货了”。我照例回复了一句“了解了”打算结束这次对话，不料这时老板突然又说了一句——

“怎么都喜欢买这一张？”

我在屏幕前不禁笑了出来，“我在逛的这家淘宝店的老板可能是个话唠哦”，我和舍友开着玩笑，一边回复了一个笑哭的表情，心里想着，或许那句“老板人真好”是这个意思吧。

老板确实有点自来熟，一下子展开话势，和我畅聊起来，讲述着《No Title-》人气之高，又抱怨起了れをる的其他专辑推力不足、买这张专辑的大多是女生、表示自己太老了听不惯等等琐碎。这个过程里，我大多数时候只是友善地附和着，但老板还是兴致盎然……真是有趣的人，我这么想着，确实因此对这家店铺产生了不少好感，更让我觉得奇妙的是，向来内向寡言的我，居然和一个陌生人相谈甚欢，虽然严格来说是被动地加入话局，但对于鲜有如此经验的我来说，不失为一次颇为神奇有趣的经历。

这样的交流，这种完全发生在两个陌生人之间的谈话，在这个世界上每天会发生成千上万次吧。我所经历的不过是一件再平凡不过的事，但这其中所蕴含的真实的温暖，却让我倍感亲切。这种独特的体验，仿佛无数个散落的点中，有两个点之间划上了一条线段，形成了一次连结。只要偏差一毫，两人大概率就永远不会有交集，但就是这一毫，赋予了各自的人生一次奇妙的际遇。霎时，我顿觉世界之渺小，在所谓人际这样无比厚重的事物面前，任何距离都可能显得微不足道。

我想起自己在 Windows 的 Microsoft Store 里偶然邂逅的一款日记应用——Choshim Diary，极其小众而冷门，但当我打开它进入其中时，却发现这样一个渺小的应用里也赫然存在自己的社区。人们在里面交换着自己的喜怒哀乐，相互之间的沟通洋溢着温情。我惊叹了，庞大的虚拟世界里，这样一个卑微而无人知晓的一隅，人们也在努力地去构建人际之间的连接，就好像你挖开一面巨大的墙面，在墙角里看见一处蚁穴，它们有着自己井然的生态，不为外界所扰。

人们是这样惧怕孤独，因而无时不刻地在与周遭发生联系。

事情的结局是，我没有在那两家店铺里的任何一家下单，我自己掏钱在日亚上购买了《No Title-》，没有中间商赚差价。不管是那位安然驻守的店主，还是那位热情和善的店主，在我以后的人生里或许再也不会出现了。我们的一生里，会有多少人像这样匆匆来去？但人类是如此平凡而孤独，以至于每一次这样的相遇都是一种伟大。

2018.3.26
